Code Ninjas BC Website

Role

Front-End Web Developer


Duration

June 2024 - July 2024


Project Overview

The purpose of this project was for my internship at Code Ninjas BC to be a part of the developer and designer team by developing front-end website pages while using and learning a JavaScript Framework and tool which is Svelte & Bun.

01 Goal

Code Ninjas British Columbia location initiated to develop a platform for their students and instructors that will be able for them to have access on managing student's level, points, and tracking their current exploration.

02 Design Process

The designer decided to use the same theme as Impact, a platform that Code Ninjas use for their coding education for kids. It has a different shade range of blue, and they use the avatar which is a Ninja. They used Illustrator for the website wireframes.

03 Development Process

As a front-end developer intern, I worked with a Full-Stack developer that guided me while learning Svelte & Bun, and giving me tips throughout the project. We used Github and Git to collaborate our coding efficiently while maintaining code organization, and Notion for tracking deadlines. Built website pages that are interactive and user-friendly as the students will be young therefore ensuring that they will be able to navigate throughout the website pages easily.

Project Insights

Adapted new tools and consistently giving updates with the Project Manager that helped expand my knowledge. Working closely with a development team in real world industry have helped hone my experience with technical and also communication skils which helped me prepare for my career. Gained deep understading with CSS and SCSS and ensuring global styling are applied for consistency.
